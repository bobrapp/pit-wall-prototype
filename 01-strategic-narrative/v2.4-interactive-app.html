Step 05 · The long lap

Monthly S&OP cycle.

Six stages. Six lanes. Three state gates. Agents handle the mechanical prep — refresh, anomaly packs, narrative drafts. People handle the things that matter — conflict, dissent, scenario choice, commitment.

Lane Model run Analyst review SME challenge Pre-S&OP Exec S&OP Post-cycle review
Data & model Gate 1Ingest internal + external signals, run models, emit quantiles + baseline forecast set. Refresh compare views and anomaly packs. Attach challenge evidence, re-score confidence. Gate 2Freeze review-package candidate. Gate 3Promote selected scenario to plan-of-record. Closed lapCompare actuals to forecast and overrides.
Analysts Receive task queue. Use Workbench + Excel to validate and override. Respond to SME challenges. Prepare consensus recommendation. Support live decisions. Review override value-add and feature gaps.
SMEs Heads-up in Teams. Optional early comments. Use Teams / Glean / SharePoint to challenge or enrich. Review conflict summary. Attend specific decision items only. Flag missed signals for next cycle.
Leadership No action. Monitor status via Morning Brief. See top disputes forming. Receive pre-read in Outlook + PBI. Approve plan-of-record + scenario choices. Review accuracy + process health.
Ops / IT Monitor freshness + dependencies. Resolve pipeline issues fast. Protect against stale-state publishing. Confirm release readiness. Support live refreshes if needed. Log incidents and improve SLAs.
GovOps Observe agent health. Track prompt + tool lineage. Monitor override + challenge patterns. Ensure evals + guardrails intact. Log final decisions + artifact lineage. Promote lessons into prompts / memory / tests.

Andon stop-the-line for forecasts

When a forecast number drifts past tolerance, the lane stops. Stale data does not get published; the wall lights amber until a human accepts or fixes it.

Step 08 · The delighters

Podium finishes.

Three modes the Pit Wall ships with on day one — plus three we hold back for v2.1. The goal is simple: make co-working in NAMA feel like the best Sunday afternoon you'll have all week.

Pole · gold

Race Mode

The 15-minute clock takes over the screen. Telemetry strip becomes a lap timer. Everyone on the wall sees the same countdown. When the chequered flag drops, the new number ships to all surfaces.

  • Lights-out countdown to start
  • Shared real-time clock across all seats
  • Chequered-flag commit moment
P2 · silver

Garage Mode

Deep analyst workspace. Lights dim, telemetry compresses, the quantile fan takes the screen. Cmd-K opens the agent. No notifications unless the wall calls you back.

  • Compare-mode hotkeys
  • Notification quiet mode
  • Auto-save staging snapshots every 30s
P3 · bronze

Podium Replay

After every decision, the system replays the lap. Every prompt, every challenge, every override — scrubbable, shareable. Great for onboarding, post-mortems, and audit.

  • Scrubbable timeline of any forecast commit
  • One-click share to onboarding
  • Auto-archived in HIBT
Coming in v2.1

Second-Opinion Agent

One click and a second NAMA-certified agent re-runs the analysis with a deliberately different prompt and source mix. If they disagree, the wall lights amber and routes to a SME — adversarial collaboration as a service.

Coming in v2.1

Ghost Lap

Overlay last cycle's forecast on this cycle's chart. See exactly where you got better, where you drifted, and where the same SME challenge keeps coming back unresolved.

Coming in v2.1

Championship Table

Lighthearted leaderboard for override value-add, challenge-to-commit conversion, agent eval lift. Not a performance review — a way to celebrate the engineers who consistently improve the plan.
