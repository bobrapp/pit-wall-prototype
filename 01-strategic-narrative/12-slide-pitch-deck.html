PILLAR 01

Planning fabric

PILLAR 02

Choose your seat

Leadership, Analyst, SME, Ops, GovOps — each seat keeps the surfaces they already love. The workspace reframes; the substrate does not. Decision rights enforced at the API.

The architecture · 03

Four layers. One brain.

An anti-fragmentation operating stack. Every surface is a client of the same planning fabric. Every action transacts through the same access layer. Every change emits the same events.

Layer APlanning Fabric
Databricks + Unity Catalog hold the plan-of-record substrate. Forecast sets · Number objects · Scenarios · Constraint links · Trust metadata. One version of every number, with full lineage to source.
Substrate
Layer BProcess Engine
State machines, trust scoring, approval routing, challenge workflows. Number states: draft → updated → challenged → committed → final → reviewed. Every action emits replayable lineage.
Engine
Layer CAccess API + Events
The anti-fragmentation layer. One API for forecasts, overrides, comments, approvals, evidence, audit. Event bus for anomalies, stale data, challenge raised, plan promoted, drift detected. Role-aware decision rights.
Contract
Layer DPersona Surfaces
Outlook · Teams · Slack · Excel · SharePoint · Power BI · Glean · Forecast Workbench · Agent Studio + HIBT. Each surface speaks its native idiom — but writes back through the same access layer.

The recurring cycle · 07

Long lap: the monthly S&OP.

Six lanes, three state gates. Agents handle the mechanical prep — refresh, anomaly packs, narrative drafts. People handle the things that matter — conflict, dissent, scenario choice, commitment.

Gate 01 · Cyan

Analyst-ready

Freshness validated. Baseline run complete. Anomaly pack generated. Compare views built.

  • Databricks pipelines green
  • Quantile fan refreshed
  • Risk-classified anomalies
  • Workbench task queue routed
Gate 02 · Amber

Challenge-ready

Consensus draft built. Challenge window open. Evidence routed. SMEs engaged. Conflict surfaced.

  • Open challenges with reason codes
  • Evidence attached to Number Objects
  • Conflict summary published
  • Scenario lab populated
Gate 03 · Champagne

Plan-of-record

Leadership commits selected scenario. All surfaces inherit committed values. HIBT entry final. Replay archived.

  • Scenario promoted to POR
  • Exec brief signed and broadcast
  • HIBT entry logged immutable
  • Post-cycle accuracy review
